## 发布

> 默认是将 Makefile 同级目录 dist 文件夹下的所有文件发布到服务器上指定目录

```bash
# 预览发布详情（不会执行发布）
make preview-dev

# 发布到测试环境
make publish-dev

# 更多命令
make help
```
